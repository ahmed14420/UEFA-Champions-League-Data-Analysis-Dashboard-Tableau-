# ⚽ UEFA Champions League Data Analysis & Dashboard


---

## 📌 Overview

This project explores historical UEFA Champions League data to uncover meaningful insights about players, clubs, and competition performance.

The goal is to transform raw data into clear, actionable insights through data cleaning and interactive visualization.

---

## 🎯 Objective

* Analyze player and club performance
* Identify historical trends and records
* Build an interactive dashboard for data exploration
* Present insights in a clear and visual format

---

## 📊 Dashboard Insights

The dashboard answers key analytical questions:

* Who are the top players by appearances and goals?
* Which clubs dominate the competition historically?
* Who are the most experienced coaches?
* Which countries have the most titles?
* What are the all-time records in a single season?
